# Excalidraw with Encrypted Cloud Backups

A self-hosted Excalidraw stack with live collaboration, MongoDB persistence, and encrypted daily backups to Backblaze B2 using restic.

## Quick Start
1. \`cp .env-example .env\`
2. Edit \`.env\`
3. \`docker compose up -d\`
4. Visit http://localhost:8080

See the code for full documentation.
